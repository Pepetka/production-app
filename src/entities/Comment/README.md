## Сущность комментария

#### Public api

- Components

`CommentForm` - компонент с формой для добавления комментария

`CommentList` - компонент со списком комментариев

- types

`Comment` - тип, описывающий комментарий

`CommentFormSchema` - тип, описывающий поля формы для добавления комментария

- slice

`commentFormReducer` - reducer, отвечающий за изменение части состояния приложения, относящейся к форме для 
добавления комментария
